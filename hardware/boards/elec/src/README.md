# Requencer — Circuit Source

Atopile hardware description files organized into circuits, boards, and auto-generated validation.

## Directory Structure

```
elec/src/
├── system.ato              ← Auto-generated validation build (DO NOT EDIT)
├── parts/                  ← Auto-generated passives (Resistor, Capacitor instances)
├── components/             ← Custom part definitions (symbols, footprints, 3D models, docs)
├── circuits/               ← Reusable circuit modules
│   ├── board-connector/    ← 2×16 header interface between boards
│   ├── button-scan/        ← 5× 74HC165D shift register chain
│   ├── dac-output/         ← 2× DAC80508 + 5× OPA4171A output stage
│   ├── display/            ← ST7796 display documentation (no physical parts)
│   ├── input-protection/   ← Voltage divider + Schottky clamp
│   ├── io-jacks/           ← 26× WQP518MA mono jacks
│   ├── led-driver/         ← 3× IS31FL3216A I2C LED drivers
│   ├── mcu/                ← PGA2350 MCU block
│   ├── midi/               ← TRS Type A MIDI IN/OUT
│   └── power/              ← Eurorack power supply
└── boards/                 ← Top-level board modules
    ├── control/            ← Control board (user-facing THT)
    └── main/               ← Main board (MCU + analog + power)
```

```
system.ato
```

## Build Entries

Defined in [`ato.yaml`](../../ato.yaml):

| Build | Entry Point | Purpose |
|-------|-------------|---------|
| `control` | `boards/control/control.ato:Control` | Control board netlist |
| `main` | `boards/main/main.ato:Main` | Main board netlist |
| `system` | `system.ato:System` | Cross-board validation (auto-generated) |

## Conventions

- **Import paths** are relative to each `.ato` file's location
- Parts imports use `../../components/<Part>/<Part>.ato` (from circuit/board dirs)
- Cross-module imports use `../../circuits/<name>/<name>.ato` (from board dirs)
- `system.ato` is regenerated by `scripts/build/gen_validation.py` — never edit manually
- `parts/` subdirectory contains auto-generated passive components (Resistor, Capacitor)
