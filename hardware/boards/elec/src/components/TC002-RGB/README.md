# TC002-RGB — RGB LED Tactile Switch

## Overview
Through-hole tactile switch with integrated common-anode RGB LED (Well Buying TC002-N11AS1XT-RGB). Combines a momentary SPST switch with an illuminated button cap.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Package   | Through-hole |
| LED       | Common anode RGB |
| Switch    | SPST momentary |
| LCSC      | C5765888 |
