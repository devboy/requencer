# TactileSwitch — Momentary Pushbutton

## Overview
Small SMD tactile switch (C&K PTS645). Used as BOOTSEL button to pull BS pin to GND for firmware programming mode.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Package   | SMD (PTS645) |
| Type      | SPST momentary |
| LCSC      | C221880 |
