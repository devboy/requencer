# TactileSwitch — Momentary Pushbutton

## Overview
Small SMD tactile switch (XKB TS-1187A-B-A-B). Used as BOOTSEL button to pull BS pin to GND for firmware programming mode.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Package   | SMD 5.1x5.1mm, 1.5mm height |
| Type      | SPST momentary |
| LCSC      | C318884 (JLCPCB basic) |
| MPN       | TS-1187A-B-A-B (XKB Connectivity) |
