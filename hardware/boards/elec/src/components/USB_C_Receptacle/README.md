# USB_C_Receptacle — USB Type-C Connector

## Overview
USB Type-C receptacle (mid-mount) for USB 2.0 device mode. Only D+/D-, VBUS, GND, CC1, CC2, and shield pins are wired; no USB 3.0 lanes. Used for firmware programming and debug.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Type      | USB-C receptacle, mid-mount |
| USB Mode  | 2.0 device only |
| LCSC      | C165948 |
