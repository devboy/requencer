# ShroudedSocket2x16 — 2x16 Shrouded Pin Socket

## Overview
32-pin (2x16) shrouded through-hole pin socket (female IDC), 2.54mm pitch. Keyed to prevent reverse insertion. Mates with ShroudedHeader2x16 on the main board.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Pitch     | 2.54mm |
| Pins      | 32 (2x16) |
| Type      | Through-hole, shrouded, keyed, female |
| LCSC      | C2685037 |
