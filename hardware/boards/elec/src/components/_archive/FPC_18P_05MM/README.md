# FPC_18P_05MM — 18-Pin 0.5mm FPC ZIF Connector

## Overview
Bottom-contact FPC ZIF connector, 18-pin, 0.5mm pitch. Designed to mate with the JC3248A035N-1 display FPC ribbon cable.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Pitch     | 0.5mm |
| Pins      | 18 |
| Contact   | Bottom |
| LCSC      | C262657 |
