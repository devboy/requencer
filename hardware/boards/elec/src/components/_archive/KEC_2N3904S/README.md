# KEC 2N3904S — NPN Transistor (Archived)

## Overview
KEC 2N3904S-RTK/PS NPN transistor in SOT-23. Was used for clock/reset output buffers.

**Note:** KEC uses a non-standard SOT-23 pinout (1=Emitter, 2=Base, 3=Collector) unlike the JEDEC standard (1=Base, 2=Emitter, 3=Collector).

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Package   | SOT-23 |
| LCSC      | C18536 (JLCPCB extended) |
| MPN       | 2N3904S (KEC) |
| Pinout    | 1=E, 2=B, 3=C (non-standard) |
