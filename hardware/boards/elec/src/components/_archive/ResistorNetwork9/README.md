# ResistorNetwork9 — 9-Pin SIP Bussed Resistor Network

## Overview
Bourns 4609X-101-103LF, a 9-pin SIP (single inline package) bussed resistor network containing 8 resistors sharing a common bus pin. 10k ohm, through-hole.

## Why This Part
Bussed network provides 8 matched pull-up or pull-down resistors with a single common connection, reducing wiring complexity. SIP package is easy to hand-solder and takes minimal board width.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Package   | SIP-9, through-hole |
| Configuration | 8 resistors, bussed (common pin 1) |
| Resistance| 10k ohm |
