# EurorackPowerHeader16 — 16-Pin Eurorack Power Connector

## Overview
Standard Doepfer 16-pin (2x8) shrouded through-hole power header, 2.54mm pitch. Carries +12V, -12V, +5V, GND, and optional CV/gate bus signals from the eurorack power bus.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Pitch     | 2.54mm |
| Pins      | 16 (2x8) |
| Type      | Through-hole, shrouded, keyed |
| LCSC      | C917645 |

## Pinout
Standard Doepfer 16-pin: pins 1-2 = -12V, pins 3-8 = GND, pins 9-10 = +12V, pins 11-12 = +5V, pins 13-16 = CV/gate bus.
