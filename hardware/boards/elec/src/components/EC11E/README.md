# EC11E — Rotary Encoder with Push Switch

## Overview
Alps EC11E series through-hole rotary encoder with quadrature output (A/B channels) and integrated momentary push switch. Standard 6mm shaft, widely used in eurorack modules for parameter adjustment.

## Why This Part
Industry-standard rotary encoder with good tactile detent feel. Alps quality at reasonable cost. The integrated push switch adds a select/confirm action without needing a separate button. Footprint is compatible with common eurorack knobs.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Package   | THT, 6mm shaft |
| Output    | Quadrature (A/B) + push switch |
| Pins      | A, GND, B, SW1, SW2 |
| LCSC      | C470069 |
