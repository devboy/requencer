# 2N7002 — 60V 300mA N-Channel MOSFET

## Overview
Logic-level N-channel MOSFET in SOT-23 package. Vgs threshold 1-2.5V allows direct 3.3V drive. Used for LCD backlight switching.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Package   | SOT-23 |
| Vds       | 60V |
| Id        | 300mA |
| Vgs(th)   | 1-2.5V |
| LCSC      | C8545 |
