# PJ366ST — 3.5mm TRS Stereo Jack

## Overview
QingPu WQP-PJ366ST 3.5mm stereo (TRS) jack, vertical through-hole mount. Same Thonkiconn form factor as the mono WQP518MA but with tip, ring, and sleeve contacts. Unswitched.

## Why This Part
TRS jack required for MIDI TRS Type A standard (tip=data, ring=+V, sleeve=shield). Same physical footprint as the mono jacks used elsewhere on the panel, maintaining visual consistency.
