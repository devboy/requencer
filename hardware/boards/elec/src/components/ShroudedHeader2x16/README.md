# ShroudedHeader2x16 — 2x16 Shrouded Pin Header

## Overview
32-pin (2x16) shrouded through-hole pin header, 2.54mm pitch. Keyed to prevent reverse insertion. Male side of the board-to-board connector between control and main boards.

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Pitch     | 2.54mm |
| Pins      | 32 (2x16) |
| Type      | Through-hole, shrouded, keyed |
| LCSC      | C2939229 |
