# 2N3904 — NPN Transistor

## Overview
General-purpose NPN transistor in SOT-23 package. Used for clock output buffer (3.3V to 5V gate level shifting).

## Key Specifications
| Parameter | Value |
|-----------|-------|
| Package   | SOT-23 |
| LCSC      | C20526 (JLCPCB basic) |
| MPN       | MMBT3904 (Changjiang/CJ) |
